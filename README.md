# План расходов
Веб версия браузерного расширения https://artemomskartemomsk-wq.github.io/Plan-PWA/
Расширение можно скачать в файлах архив(План расходов расширение для браузера.zip).

Простое приложение, чтобы понять, сколько реально нужно откладывать с каждой зарплаты, чтобы покрыть все регулярные траты.

Показывает:
- сколько откладывать с каждой зарплаты (1 зарплата в месяц)
- общую сумму всех постоянных расходов за год

## Для чего

- Считает реальную нагрузку от подписок, аренды, кредитов, алиментов, кружков детей и других регулярных платежей
- Помогает быстро увидеть, сколько денег «съедают» постоянные траты
- Удобно планировать бюджет и понимать, остаётся ли что-то на жизнь и накопления

## Как пользоваться

1. Открываете приложение (по ссылке или установив расширение)
2. Добавляете свои регулярные расходы:

   - Выбираете периодичность: ежедневно / еженедельно / ежемесячно / ежегодно
   - Пишете короткое описание (например: «Netflix», «Аренда», «Кредит», «Йога»)
   - Указываете сумму

3. Приложение сразу пересчитывает:
   - Сколько нужно откладывать с каждой зарплаты
   - Итоговая сумма трат за год

4. Можно:
   - Редактировать запись (клик по строке)
   - Удалять (крестик справа)
   - Экспортировать все данные в JSON
   - Импортировать ранее сохранённый файл

## Данные хранятся

- В расширении Chrome — синхронизируются между устройствами (если включена синхронизация)
- В Web-версии — локально в браузере (localStorage)(после сброса кеша очищается)

Готово. Никаких регистраций, подписок и рекламы.

Приятного планирования!
